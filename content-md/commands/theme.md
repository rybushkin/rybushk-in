# Command: theme

Команда `theme` позволяет изменить цветовую схему сайта.

## Description

Change color, not personality.

## Available Themes

Список доступных тем (отображается при вызове команды без аргументов):

-   • classic
-   • retro
-   • cyberpunk
-   • ubuntu
-   • gameboy
-   • excel
-   • norton
-   • monochrome
-   • tron

## Usage

Usage: theme <name>

## Example

Example: theme cyberpunk
